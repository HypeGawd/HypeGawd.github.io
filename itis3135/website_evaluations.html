Website Evaluations

Airbnb

  • Contrast: The Airbnb website makes excellent use of contrast, with clear distinctions between different sections of the site.
  • Repetition: The use of repetition on the Airbnb website is evident in the consistent use of fonts, colors, and design elements.
  • Alignment: The alignment on the Airbnb website is precise, with all elements being aligned to a grid. This creates a sense of order and makes it easy for the user to navigate the site.
  • Proximity: The Airbnb website uses proximity effectively, grouping related elements together, and creating a clear visual hierarchy.

Arngren

  • Contrast: The Arngren website lacks contrast, with many elements competing for attention.
  • Repetition: There is no clear repetition on the Arngren website, with different fonts, colors, and design elements used throughout. This lack of consistency makes the site feel disorganized and confusing.
  • Alignment: The alignment on the Arngren website is haphazard, with elements appearing to be randomly placed on the page. This lack of order makes the site difficult to navigate and hard to read.
  • Proximity: The Arngren website uses proximity poorly, with unrelated elements grouped together and little visual hierarchy. This creates a cluttered and confusing design, making it challenging for users to find what they are looking for.
